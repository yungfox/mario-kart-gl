# mario-kart-gl

Recreation of a Mario Kart 64 item box with C and OpenGL.
Runs on macOS, Linux and Windows thanks to [RGFW](https://github.com/ColleagueRiley/RGFW).

## Dependencies

- CMake
- OpenGL 3.3

## Quick start

```console
$ cmake -B build
$ cd build

# posix
$ make
$ ./mario-kart-gl

# windows
$ cmake --build .
$ .\Debug\mario-kart-gl.exe
```
